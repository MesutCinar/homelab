# Video title
Set up your homelab - part 4 - mailpit

# Video description

In this session we will add mailpit as a development tool to the NAS homelab.

https://hub.docker.com/r/axllent/mailpit

You can test the server with a Powershell on Windows:

Send-MailMessage -From 'me@lab.com' -To 'you@lab.com' -Subject 'Testmail' -Body 'Hello world.' -SmtpServer 'nas.local' -Port 1025
